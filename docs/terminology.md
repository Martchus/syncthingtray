# Terminology
Various tools are available for Syncthing. The following terminology is used throughout this documentation:

* "Syncthing", "Syncthing itself" or "Syncthing backend": The continuous file synchronization program available
  at https://github.com/syncthing/syncthing.
* "Syncthing Tray": The repository at https://github.com/Martchus/syncthingtray, which provides various utilities
  and integrations for "Syncthing". It can be built so that the resulting artifacts (executables, APKs, etc.)
  bundle "Syncthing". The official builds for Android, Windows, and GNU/Linux bundle "Syncthing". "Syncthing Tray"
  is also the name of the primary system-tray-focused integration this repository provides.
* "Syncthing App": The name of the app produced when compiling "Syncthing Tray" for Android.
* "Syncthing Plasmoid": The name of the KDE Plasma Applet (or "Plasmoid") produced when compiling "Syncthing Tray"
  for GNU/Linux with KDE integrations.
* "syncthingctl" or "Additional command-line client for Syncthing": A command-line client for "Syncthing" provided
  by "Syncthing Tray".
* "Syncthing-Fork": An app that was maintained and developed by Catfriend, which also bundles "Syncthing". It is
  no longer maintained by Catfriend. Mentions throughout this documentation always refer to the version from Catfriend unless stated otherwise.
